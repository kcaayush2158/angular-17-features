Angular 17 adds a new template syntax for adding control flow statements in your Angular templates without using *ngIf, *ngFor or *ngSwitch.




If Statement

@if (loggedIn) {

Rendered conditionally!

} @else {

The user is not logged in

}

Switch statement

@switch (source) { @case ("google") {

Thanks for finding us on Google

} @case ("youtube") {

Thanks for following us on YouTube

} @default {

Maybe we'll meet in the future

} }

Loop


@for (item of items; track item.id) {

Framework name: {{ item.name }}


}@empty{

Hello world
